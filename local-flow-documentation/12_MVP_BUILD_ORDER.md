# 12 — MVP Build Order

If you want the fastest working version, build only these phases first.

## MVP Phase Order

1. Phase 0 — Project Foundation
2. Phase 1 — Tray App
3. Phase 2 — Global Shortcut
4. Phase 3 — Recording State Machine
5. Phase 4 — Floating Overlay
6. Phase 5 — Microphone Detection
7. Phase 6 — Audio Recording
8. Phase 7 — STT Service
9. Phase 8 — Recording to Transcription
10. Phase 9 — Clipboard and Paste
11. Phase 10 — Ollama Integration
12. Phase 11 — Basic Cleanup Mode
13. Phase 16 — Settings Page
14. Phase 19 — Error Handling
15. Phase 21 — Installer

## What This MVP Gives You

After these phases, the app should:

- Run on Windows
- Stay in tray
- Use global shortcut
- Record voice
- Transcribe locally
- Optionally clean with Ollama
- Paste into active app
- Save settings
- Handle common errors
- Build as installer

## What to Avoid Before MVP

Avoid:

- Real-time streaming transcription
- Auto-silence detection
- Complex voice commands
- Cloud sync
- Multi-user support
- Plugin system
- Mobile app
